LIBRARIAN ACTIVITY

 

The librarian activity creates the basic LEGOCAD user libraries and allows to create, modify or copy a module.

The basic libraries are:

 

libut

user process library

libut_reg

library of control schemes

libreg

control modules library

 

This activity also allows to modify the associated module icons through Edicole tools.

Librarian: create or modify a module

 

Creation (or modification) of legocad process modules is provided by a librarian's tool called New Mod.

The New Mod tool provides some interactive utilities to create a new module or to modify modules that already exist and have been previously created by New Mod.

If a model was created with an older version of LEGOCAD, New Mod will not be able to modify the module and, the user will have to edit the fortran code directly.

 

Creating a new LEGOCAD module involves the following step:

 

1) Selecting from New Mod pulldown menu Module configuration | input/output variables set-up a dialog is displayed by the tools to configure the basic information of the module i.e. :

 

Number of state output variables

Number of algebraic output variables

Number of input variables

Number of geometrical data

Type of Jacobian computation.

 

2) After the module set-up New Mod displays in the "New Mod definition" window a series of input forms where to insert descriptions, fortran symbols and normalise values for all variables and for defined geometrical or physical data.

 

3) Using menu selection Module configuration | Jacobian matrix topology you define which variables are used in which equations.

 

Information provided in step 1) 2) and 3) are automatically converted during fortran code generation (menu selection File | Save and generate FTN).

The user generally have to complete code sections constituting LEGOCAD process module code: a series of dialog window are provided for the insertion of user code or additional data (N.B. user code can be added only through dialog window provided; fortran coded directly on module fortran file will be overwritten by fortran automatic generation).

The sections have the following inputs:

 

Data

section (I2)

added variable declarations

added code

Jacobian

section (JC)

added variable declarations

added code

jacobian matrix coefficient

Residual

section

added variable declaration

added code

residual

Equation

section (D1)

equation descriptions

m.u.

normalization constant

 

At the end of module construction the user will be able to write the fortran code from the File's pulldown menu (menu selection File | Save and generate FTN).

Librarian: copy a module

 

The user can copy a module to his private library from main window pulldown menu by using "Copy From" . The modules are copied from another user (for each library) or from standard libraries. These libraries are :

Standard library (only for libreg library);

Auxiliary standard library, Nuclear standard library and Special standard library (only for the libut library)

When the user choose the "Different user " option, Librarian will display a window named "Module copy" with the filter utility to copy the desired files. Otherwise the same window "Modules list" will be display to select the module from a standard list. This page can carry out three kind of transfers:

1 only the module fortran code

2 only the associated module icons

3 both fortran and icons.

 

Information on the page layout is available in:

1 Modules list page

 

Librarian: delete a module

 

The user can delete a module by selecting it on the scrolled list and clicking on the Delete's Module pulldown menu. A confirmation dialog box will be displayed.

 

Librarian: create or modify associated module icons

 

The user can create new associated icons or modify already existing ones through Edicole tool by calling the Icons's pulldown menu.

Working with the displayed window "Lego icon editor", the user can select the desired icon among the existing ones either change it or create a new using the standard xbitmap utility .

 

Librarian: Create library

 

When the user has finished module (and related icons) setting, he will be able to create the selected library by using Compiling Library pulldown menu.

Librarian layout: New Mod set-up page

 

The New Mod set-up page is a dialog page which contains some text fields to receive basic set-up numbers; these numbers are :

 

Number of state output variables

Number of algebraic output variables

Number of input variables

Number of geometrical data

 

The user can also set the calculation with numerical or analytical jacobians.

 

The OK and Cancel push button complete the page.
