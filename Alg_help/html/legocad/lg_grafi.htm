GRAPHICS ACTIVITY

 

The Graphics tool allows to manage a graphic trend of max. 4 variables recorded during simulation (simulation historical data are recorded on F22 file).

Using Graphics the user select variables and any other parameter such the units of measurements, upper and lower measurement limits etc.

Furthermore the user can save variables and their specific parameters in groups which can be called either during the same or during the next execution of the program.

All samples stored in the file (i.e. the whole simulation) are displayed on the page.

It is also possible to zoom on a particularly drawing area and to use the collimation function to know variables values of all the pixels displayed.

Displaying is static if simulation is not active, while trends are compressed during the simulation to make room for the new samples.

2 Select displaying variable set

3 Use groups

4 Use zoom

5 Change m.u.

6 Change displayed full scale

Graphics: open file

 

Graphics activity in LEGOCAD calls the GRAPHICS tool with F22 task output file already opened by default .

The user can also open another data file by selecting File Open's pulldown menu from the main window; Graphics will display the file selection page.

 

Information on these pages layout is available in:

1 File selection page layout

 

Graphics: select variables

 

The user can display any recorded variables by selecting them in a displayed list on the selection variable page which is opened from the pulldown menu (Measure/Selection). up to four variables are displayed simultaneously.

The page displays variables which are selected in the list; the selection carries the variable in a position fixed by the user by clicking one of the four "location" toggle buttons. In this way the user can also choose the relative order of every variable in the graphic page. This operation is made by clicking the "location" toggle button and then selecting the variable. A new selection deletes an older one.

Other available operations are the deletion of a selection and the search for one variable in the list. In particular, when a name is searched for in the list the "Previous" and "Near" push buttons can move the search result in the list. Generally these push buttons scroll the list.

 

Information on these pages layout is available in:

1 Variables selection page layout

 

Graphics: use groups

 

A group is a collection of up to four variables (and their parameters) which is saved in a file to ease future selection. The user can store a group and give it a name which will be used for new selections.

Every variable can belong to more than one group.

Groups can be deleted.

The user can set groups from the pulldown menu (Graphic/Selection, Graphic/Save) by calling dedicated pages.

 

Information on these pages layout is available in:

1 Groups pages layout

 

Graphics: use zoom

 

To have a better view of the line displayed the user can use the zoom function which is available either for X-axis (from Graphics Zoom pulldown menu) or both XY axis (from Default | On-Off ordinate zoom pulldown menu)

The user disables the zoom function by pressing using the Defaults | Restore push button from the pulldown menu.

 

Graphics: change m.u.

 

Through this function each variable can be related to a default unit of measurement chosen among those provided.

This is a permanent choice, which will remain unvaried in the next executions of the program. All the variables which will be selected at a later time will use the default values. The user can also change the join unit of measurement associated with the variable.

So there are two dialog pages to change the unit of measurements.

 

Information on this page layout is available in:

1 M.u. selection pages layout

 

Graphics: change upper and lower scale limits

 

The user can change the full scale of a displayed variable and the scaling modality. By default Graphics uses autoscaling modality (scale limits are calculated so that trends are drawn in all whole drawing area). However the fixed scale can be set giving maximum and minimum values for each variable.

Graphics layout: file selection page

 

The file selection page has four text fields that contain the pathname of a file in the "f22.dat" format. This page has also four toggle buttons related to the file to be opened.

Although the file pathname could already exist the user can insert a new one. These files have previously been inserted from the user (and saved by Graphics) and the selection is made by clicking on the left toggle button. An "OK" push button confirms the choice and closes the page whereas a "Cancel" push button quit the selection.

 

Graphics layout: variables page

 

This page has four "location" toggle buttons, a scrolled list containing the graphical variables list and five "operation" push buttons.

When the mouse left button is clicked on a variable of the list, the variable will be displayed in a label to the right of the chosen toggle button.

 

At the bottom of the page there are the following push buttons:

OK

Confirms all selections and closes the page

Quit

Exits from the page

Next

Scrolls or searches forward

Previous

Scrolls or searches back

Cancel

Deletes a selection

 

 

Graphics layout: groups pages

 

Groups can be used through the pulldown menu. In particular it is possible to

save a group

using Group saving page

select a group

using Group selection page

 

Group save page layout

This page displays a scrolled window containing the list of the names of the related to the saved groups, a text field to insert the name of a new group and two push buttons to confirm or to quit the page. The user can insert a new group by selecting an available one and inserting its name in the text field.

 

Group select page layout

Likewise group selection page has the same list in a scrolled window. The user can select the group by clicking on a group name with the first mouse button and then by pushing on the "OK" button. The "Quit" push button closes the page without any selection.

 

Graphics layout: m.u. pages

 

Available unit of measurement pages :

 

Default unit of measurement page (from Default/M.u. p.m.)

This dialog page can set the default values for the unit of measurement of every kind of variable. The page has a scrolled window containing the different kinds of variables and below toggle buttons. After selecting one of them, the user sets one of the available m.u. to the chosen variables.

The confirmation area has "OK" and "Cancel" push buttons.

 

Changing unit of measurement page (from Measure/Change M.u. p.m.)

Through the unit of measurement page the user can select a different m.u. for one of the displayed variables.

The page has four toggle buttons to set the variable which could be modified and a number of toggle buttons equal to the available m.u..

The confirmation area has "OK" and "Cancel" push buttons.

 

Graphics layout: scale limits page

 

The layout of the scale limits page includes the:

 

Selected variables area:

featuring the label of the four (maximum) variables that have been selected and their toggle buttons to be able to modify a specific variable.

 

Drawing modality area:

featuring the autoscaling and fixed scale toggle buttons as well as the Max/min text field to insert the maximum and minimum values of the variable (in the fixed scale modality). Once the user pressed enter on the keyboard after insert operation the page will close and the entry saved.

 

Confirmation area:

featuring the OK and Quit push buttons. The first push button saves the newly inserted parameters and closes the page while, the second will close the page without saving.
