Example of a LEGO model building:

modules selection

 

For the representation of this system we are going to use three LEGO modules of the LEGOCAD process library:

1) module CPOM: centrifugal water pump;

2) module VALV: generic steam-water valve;

3) module COLT: steam-water manifold (this module is defined in this model to take into account the storage phenomena in the tube between the pump and the valve).

Each module has to be istanciated only once in the model, so the number of blocks in the model is three.

When the modules CPOM and VALV are istanciated in the model, the LEGOCAD "Topology" program asks the user which output variable he want to define between inlet pressure, outlet pressure and water flow rate. In the same way, when the user defines the module COLT, the program asks him to define the number of inlet and outlet branches. The user must operate correct definitions to be able, in a second time, to connect appropriately the three blocks defined in the model.

 

Example of a LEGO model building:

list of block variables

 

For each block we have the following variables:

 

Block POMP:

 

nput variables:

PUPMPOMP = pressure at the pump delivery

PIPMPOMP = pressure at the pump suction

HIPMPOMP = water enthalpy at the pump suction

OMPMPOMP = angular velocity of the pump

NPOMPOMP = number of identical pumps in parallel configuration

 

Output variables:

HUPMPOMP = water enthalpy at the pump delivery

WPOMPOMP = pump total capacity

 

Block TUBO:

 

Input variables:

WE_1TUBO = tube inlet water flow rate

HE_1TUBO = tube inlet water enthalpy

WU_1TUBO = tube outlet water flow rate

HU_1TUBO = tube outlet water enthalpy (in case of back-flow)

TESTTUBO = tube external temperature

GESTTUBO = tube external heat exchange coefficient

 

Output variables:

HCOLTUBO = mean water enthalpy in the tube

PCOLTUBO = mean water pressure in the tube

TCOLTUBO = tube metal temperature

 

Block VALV:

 

Input variables:

PIVAVALV = valve inlet pressure

PUVAVALV = valve outlet pressure

HVALVALV = water enthalpy at the valve inlet

ALZAVALV = valve opening degree

 

Output variables:

WVALVALV = water flow rate in the valve

Example of a LEGO model building:

data assignment

 

To assign blocks geometrical and physical data, the user has to examine the documentation about the modules CPOM, COLT and VALV:

 

Block POMP:

 

TYP = 1 This parameter identifies the characteristic head-flow rate of the pump: this characteristic may be assigned by the user in the auxiliary FORTRAN of the model (the auxiliary FORTRAN contains particular subroutines that the user can modify to his own necessity). In this case the pump characteristic is unknown so we must suppose a reasonable relation between pump flow rate and pump head.

INER = 0 Moment of inertia of the pump [kg m²]. This data must be assigned only if the pump rotational speed is an output variable; in our model we have chosen a pump model with the rotational speed as input variable, so this data value is irrelevant.

QVNOM = 0.1 Design water flow rate [m³/s].

OMNOM = 314 Design rotational speed [rad/s].

DPNOM = 9.e5 Design head [Pa].

POTN = 0 Design power absorbed by the pump [W]: if the POTN data is not provided, it is automatically calculated by the module on the basis of the compulsory data QVNOM and DPNOM and ETA (efficiency) = 0.85.

DENS. = 998 Water density [kg/m³]. In this case the density is calculated for a pressure of 1 bar and a temperature of 20 °C.

N. GIRAN. = 1 Number of impellers. This value is only used for the head calculation (in meters of water column) that appears in the expression of the pump characteristic index. If this value is not provided, it is assumed N.GIRAN. = 1.

VALV INT = 0 Pump interception index. The value assigned to VALV INT can be 0 (not intercepted flow rate) or 1 (intercepted flow rate). User must use the value 1 only if OMPM = 0 and no valve is foreseen upstream or downstream the pump.

 

Block TUBO:

 

NU.TU.EN = 1 Number of inlet branches.

NU.TU.US = 1 Number of outlet branches.

VOLUME = 0.49 Volume of water inside the tube [m³]. This value is calculated in relation to the tube inside diameter and length.

MASSA ME = 637 Metallic tube mass [kg]. This value is calculated in relation to the tube inside and outside diameter, the length and the metal density.

CP MET = 630 Metal specific heat [J/(kg K)].

COND MET = 35 Metal thermal conductivity [W/(m K)]

SPESSORE = 0.01 Wall thickness [m].

SUP. ES = 8.482 Tube external surface [m²]. This value is calculated in relation to the tube outside diameter and length.

 

Block VALV:

 

ALFA = 1 Coefficient used to define the valve characteristic, that is the cross section area variation law as a function of ALZA opening degree following the formula: AREA=ALZA**ALFA. If we suppose the characteristic as linear, we must assign ALFA = 1.

CFA = 0.85 Valve recovery coefficient with fully opened valve (ALZA=1) Typical value CFA = 0.85.

COD. = 2 parameter that identifies the type of valve to simulate and so the mathematical model to use. We suppose to use a not sonic simplified model: in this case COD = 2.

CVA = 0 identification coefficient of valve working point. If this datum is not available (as in this case), the user assigns CVA = 0 and provides the following data referred to a known valve working point.

PORTATA = 100 Fluid flow rate [kg/s].

P1 MONTE = 10.e5 Inlet pressure [Pa].

P2 VALLE = 1.e5 Outlet pressure [Pa].

ENTALPIA = 83.4e3 Inlet fluid enthalpy [J/(kg K)].

CORSA = 0.8 Opening degree.

Example of a LEGO model building:

execution of a test transient

 

The steady state calculated is the starting point for any simulation we want execute on this model.

For instance, we can try to stop the pump by reducing its rotational speed from design value to zero value. Before starting the simulation we are going to define the following parameters:

 

Simulation parameters:

 

Integration step = 0.5 s

Max simulation time = 400 s

Variables values recording step = 2 s

 

Perturbed variables:

 

OMPMPOM

Type of perturbation: ramp

Start time of the perturbation = 10 s

Stop time of the perturbation = 310 s

Variable value after the perturbation = 0 rad/s
