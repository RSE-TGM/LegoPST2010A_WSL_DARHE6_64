Test case 4: general description

The following test case represents a portion of the condensate cycle of a conventional thermoelectrical power plant.

The main components of the simulated system are:

1. the condenser (block COND - module CNDN)

2. the extraction pump (block PEC1 - module CPOM)

3. the horizontal preheater (block R1 - module RSOR)

4. the vertical preheater (block R2 - module RSCD)

5. the deareator (block DEAR - module DEGN)

The other components involved in the model represent pressure drops (valves) and pipe volumes (manifolds).

The dynamical behaviour of this model has been studied in the case of a 10% reduction of the steam flow rate to the condenser and of the bleeding steam flow rate to preheaters and deareator.
