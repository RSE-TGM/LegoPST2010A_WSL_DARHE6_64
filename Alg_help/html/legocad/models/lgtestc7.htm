Test case 7: general description

The following test case represents the model of a 120 MW turbogas plant (TurboTecnica MS9000) and of the main functions performed by its control system.

The turbogas is simulated by three blocks, representing the three main components of a turbogas plant:

1. Axial air compressor (block COAF)

2. Combustion chamber (block CPAF)

3. Axial gas turbine (block TPAF)

The function performed by the simplified control system (block RETG) are:

1. to limit turbine exhaust gas temperature, adjusting the position of the compressor Inlet Guide Vanes (IGV) and the fuel flow rate.

2. to control the rotational speed of the machine

The block IRTG has been inserted in the topology of the model to interface the control system variables with the process variables.

The block MART (inertia of the turbogas) operates when the plant is not connected with the network (during start-up, shutdown, trip, load-rejection): in this case the rotational speed of the machine is not imposed by the network frequency, but it is given by the energy equation for the rotating masses.

The steady condition calculated for the model represents the "full speed no load" state of the plant.

The dynamical behaviour of this system has been studied in a case of a load ramp from 0% to 100%.

SUBROUTINE TGLINTAV(TABP,LP,TABX,LX,TABY,X,P,Y,DYX,DYP)

DIMENSION TABP(*),TABX(*),TABY(*)

LP1=LP-1

DO 10 I=2,LP1

IF(P-TABP(I)) 20,20,10

10 CONTINUE

I=LP

20 IP=LX*(I-2)+1

CALL TGLININ(TABX,LX,TABY(IP),X,Y1,DYX1)

IP=IP+LX

CALL TGLININ(TABX,LX,TABY(IP),X,Y2,DYX2)

DYP=(Y2-Y1)/(TABP(I)-TABP(I-1))

Y=Y1+(P-TABP(I-1))*DYP

DYX=DYX1+(P-TABP(I-1))*(DYX2-DYX1)/(TABP(I)-TABP(I-1))

RETURN

END
