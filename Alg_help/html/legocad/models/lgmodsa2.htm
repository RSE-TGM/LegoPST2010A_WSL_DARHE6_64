Example 2 - LEGO modules choice

The model is composed of five components. In the building of the topology of this model, the choice of the LEGO modules to be used is very easy:

the induction valve has to be simulated by the VALV module (VTUR block);

the reaction stages have to be simulated by the TREN module (STA1 and STA2 blocks);

the condenser has to be simulated by the CNDN module (COND block);

the bleeding valve has to be simulated by the SPIL module (BLED block).

Note that it would be wrong to simulate the bleeding valve using the VALV module, as a specific module has been developed to allow the simulation of steam turbine bleedings.

As the TREN module, used to simulate tre reaction stages, has the upstream pressure as output variable, and the inlet steam flow rate as input variable, the VALV module has to be istanciated using the fluid flow rate as output variable and the downstream pressure as input variable.

Like in the first example, a NODO module (HSTE block) will be added to give to the valve and to the first stage the same value for steam inlet entalpy.

If the model would have to be used to simulate turbine group startup, shutdown or load rejection transients, a NODO should be added to give to both the stages the same value for turbine shaft rotational speed. For this example we will assume that these kind of transients will not be simulated, so no other module is required.

Example 2 - variables connection

How should one connect the variables of the previously created blocks?

As we said, the steam flow rate at the 1-st turbine stage inlet has to be connected to the induction valve flow rate, and the downstream pressure of the valve has to be connected to the upstream pressure of the 1-st turbine stage. Bot for the valve and the 1-st turbine stage, the inlet entalpy has to be connected to the output variable of the HSTE adder node.

The steam flow rate and entalpy and the at the 2-nd stage inlet have to be connected to the steam and entalpy at the 1-s stage outlet (the bleeding flow rate is taken into account by the 1-st stage simulation module). On the contary, the outlet pressure of the 1-st stage (which is an input) has to be connected to the 2-nd stage inlet pressure (which is an output).

The bleeding flow rate of the 1-st stage has to be connected to the steam flow rate through the bleeding valve. As the pressure at the 1-st stage outlet is computed as inlet pressure for the 2-nd stage, the inlet pressure of the bleeding valve (which is equal to the outlet pressure of the 1-st stage) has to be connected to the inlet pressure of the 2-nd stage.

The steam entalpy and flow rate at main condenser inlet have to be connected to the steam entalpy and flow rate at the 2-nd stage outlet, and the 2-nd stage outlet pressure has to be connected to the condenser cavity pressure.
