EDIT DATA ACTIVITY

 

Edit Data allows you to assign tentative starting values for block variables, set physical and geometrical data and to define which are known variables for the mathematical solution of the system.

You may select LEGO blocks from list or from symbolic diagram (corresponding to Symbolic diagram and Only by list modalities) and load them into an editor (Values and Data editor) that display all variables and data of blocks loaded completed by input fields for values assignment.

 

HOW TO

1 Use symbolic diagram modality

2 Use only by list modality

3 Set normalization

4 Set tentative variables values

5 Set data values

6 Set known or unknown variables

Edit Data: symbolic Diagram

 

The symbolic diagram modality displays the basic page data editor page, added pages are diagram page's index and model diagram page.

The user can insert data by opening variable and data editors after selecting blocks (pulldown menu selection Edit | Data or Edit | Variables).

Blocks can be selected both in model diagram page and in the data editor page (like in the only by list modality). If variables and data editor is opened other blocks can be loaded into it selecting File | Load from its pulldown menu.

Normalization values can be modified.

 

Information on the page layout is available in:

 

1 Diagram page's index layout

2 Model diagram page layout

3 Data editing page layout

4 Normalization page layout

5 Variable values & data page layout

 

 

Edit Data: only by list

 

In only by list modality blocks are selected from the model's block list presented by the data editor page.

User can assign numeric values to block variables or to physical & geometrical parameters calling the variable and data editors while one or more blocks are selected (pulldown menu selection Edit | Data or Edit | Variables).

 

Information on page's layout is available in

 

1 Data editing page layout

2 Normalization page layout

Edit Data: variables values

 

The user can insert tentative variable values of a selected block by opening variable and data editors in the "Variables..." modality. This editor could be opened either from data editor page or from model diagram page selecting Edit | Variables from the pulldown menu.

More than one block at a time can be loaded into the variable & data editor; variable and data of all blocks loaded will be editable.

Every variable value must be inserted by the user in the text field adjacent to the variable's name label. A toggle button (one for each variable) allows to change variable qualification; if the variable must be treated as known in equation system resolution, toggle is pressed in (ON:red coloured) otherwise popped out (OFF).

For known/unknown variable balancing look at main window counters to see the total number of equation, number of unknown variables, of input variables and of known variables

An option menu (one for each variable) is provided for switching between the different units of measurement available for the variable. The value of the variable is represented accordingly to the selection in the units of measurements option menu.

 

Edit Data: data values

 

The user can insert physical data values of a selected block by opening variable and data editors in the "Data..." modality. This editor could be opened either from data editor page or from model diagram page selecting Edit | Data from the pulldown menu.

More than one block at a time can be loaded into the variable & data editor; variable and data of all blocks loaded will be editable.

Every data value must be inserted by the user in the text field adjacent to the data's name label.

 

Comment lines are also displayed.

 

Edit Data: set known variables

 

The user can set as known some variables values to obtain a number of equations equal to the number of unknown variables.

This operation is achieved setting ON (red coloured) the toggle button of the variable line (In Variable & Data Editor) and applying the operation.

On the data editor main window (Data editor window) are displayed and updated at every Apply operation the number of equation, number of unknown variables, of input variables and of known variables; look at these values for correct balancing of known/unknown variables (n. of equation must be equal to n. of unknown variables; n. of input variables equal to known variables).

Edit Data: variables & data editors

 

Variable & data editors enable the modification of physical data values or values and status (known/unknown) of block's variables selected , according to the kind of selection. The modification are saved by pushing the apply push button.

Simultaneous blocks loading is permitted.

 

Edit Data: data editor page

 

This page contains the list of the model blocks in a scrolled window with find facilities. Single or multiple selections are allowed.
