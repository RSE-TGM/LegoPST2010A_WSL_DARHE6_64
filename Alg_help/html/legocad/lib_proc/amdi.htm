MODULE AMDI CARD A

Model Generalities

 

ANALOG MULTIPLIER OF TWO SIGNALS

 

The module AMDI multiplies two analog signals. The output value is restricted between an assigned low and high limit value.

 

 

MODULE AMDI CARD B

Topological data

 

INPUT-OUTPUT SCHEME

 

 

PHYSICAL MEANING OF MODULE INPUT-OUTPUT VARIABLES

 

[…] UMOL product

[…] I1MO first factor

[…] I2MO second factor 

 

 

MODULE AMDI CARD C

Physical and Geometrical Data

 

In addition to the initial values of input and output variables, the user must assign the following data:

 

[…] U_MAX product high limit value

[…] U_MIN product low limit value

[…] NORM FA1 first factor normalization

[…] NORM FA2 second factor normalization

 

 

MODULE AMDI CARD D

Module Variables Initialization

 

To initialize the AMDI module variables no particular prescription is required.

 

 

MODULE AMDI CARD E

User's Subprograms

 

AMDI doesn’t prescribe any FORAUS subroutine.

 

 

MODULE AMDI CARD F

Residuals

 

 

RNI(1) = (UMOL-I1MO*I2MO)/(NORM FA1*NORM FA2)
