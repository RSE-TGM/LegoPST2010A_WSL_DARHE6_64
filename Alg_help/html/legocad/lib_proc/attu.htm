MODULE ATTU CARD A

Model Generalities

 

SIMPLIFIED ACTUATORS

 

The module ATTU belongs to the thermodinamic base library. It simulates the behaviour of a group of simplified actuators (maximum 7).

Each actuator is modelized as a simple first-order filter (with an user-assigned time constant) between the actuated variable and its set-point (e.g.: the stem position of a valve an tre position set-point evaluated by the control system).

 

In order to allow the correct connection with other LEGO modules (avoiding the emission of program warnings) the first character of the input and output variables can be choosen by the user.

PHYSICAL MEANING OF MODULE INPUT-OUTPUT VARIABLES

 

The first character of the input and output variables can be choosen by the user. In this case the name of the input and output variables are the following (where the "a" is replaced with the character assigned by the user):

 

[ ] aS_1, ... aS_n set-point of the variable to be actuated

[ ] aV_1, ... aV_n value set-point of the variable to be actuated

 

If the user doesn’t assign the initial character, the default names are used for input and output variables:

 

[ ] ST_1, ... ST_n set-point of the variable to be actuated

[ ] VV_1, ... VV_n value set-point of the variable to be actuated

 

 

MODULE ATTU CARD C

Physical and Geometrical Data

 

In addition to initial values of TPAF module input and output variables, user must assign the following data set to complete the module input on the file F14:

[s] TAU1...TAUn actuators time constants

[ ] GAIN1...GAINn scale factors (filter gains)

[ ] VMAX1...VMAXn maximum variation speed of the outputs

[ ] NORU1...NORUn normalization factors for the input variables

[ ] NORX1...NORXn normalization factors for the output variables

 

In order to assign the normalization factors, both for the input and the output variables of the ATTU module (required by LEGO for the normalization of the equations) the user must assign in the F14 file the following alphanumeric data:

P0 for pressures

H0 for enthalpies

T0 for temperatures

W0 for flow rates

RO0 for densities

AL0 for lenghts

V0 for volumes

DP0 for differential pressures

blank value = 1. (e.g: for valve stem positions)

 

The maximum variation speeds are assigned in the same measurement units of the related variable divided by the default time measurement unit (second).
