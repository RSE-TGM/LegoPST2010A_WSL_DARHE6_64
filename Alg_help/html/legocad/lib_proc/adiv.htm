MODULE ADIV CARD A

Model Generalities

 

SIGNALS DIVISOR

 

The module ADIV calculates the division of one analog signal into another.

Two data are provided as saturation values for the result (min and max).

 

 

MODULE ADIV CARD B

Topological data

 

INPUT-OUTPUT SCHEME

 

 

PHYSICAL MEANING OF MODULE INPUT-OUTPUT VARIABLES

 

[…] UDIV result

[…] INUM numerator

[…] IDEN denominator

 

 

MODULE ADIV CARD C

Physical and Geometrical Data

 

In addition to the initial values of input and output variables, one must assign the following data:

[…] U MAX max saturation

[…] U MIN min saturation

[…] NORM NUM numerator normalization

[…] NORM DEN denominator normalization

 

 

MODULE ADIV CARD D

Module Variables Initialization

 

To initialize the ADIV module variables no particular prescription is required.

 

 

MODULE ADIV CARD E
