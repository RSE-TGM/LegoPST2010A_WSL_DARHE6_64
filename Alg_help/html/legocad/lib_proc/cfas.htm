MODULE CFAS CARD B

Topological data

 

INPUT-OUTPUT SCHEME

 

 

PHYSICAL MEANING OF MODULE INPUT-OUTPUT VARIABLES

  

[Volt] UR00 real component of the voltage

[Volt] UX00 imaginary component of the voltage

[A] IR00 real component of the current

[A] IX00 imaginary component of the current

[p.u.] XCOS cosj between voltage and current

[deg] XVFA phase angle of the voltage

[W] QATT electrical active power

 

 

MODULE CFAS CARD C

Physical and Geometrical Data

 

In addition to the initial values of input and output variables, user must assign the following data:

[W] QBASE normalization constant for electrical power

[Volt] VBASE normalization constant for electrical voltage
