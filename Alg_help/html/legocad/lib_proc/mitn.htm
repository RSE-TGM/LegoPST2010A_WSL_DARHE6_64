MODULE MITN CARD A

Model Generalities

 

N INPUTS MIXER

 

The module MITN calculates the mixing mass flow rate and temperature of N inlet branches. In particular:

The output mass flow rate is the sum of all inlet ones.

The output temperature is an average of the inlet ones, weighed with the ratio between mass flow rate of the actual branch and the sum of all inlet mass flow rates. It’s equivalent to an enthalpy balance, supposing the specific heat constant enough for all input branches to allow its simplifying among all the terms of the balance.

In the enthalpy balance, negative inlet mass flow rates are set to zero. So, mass flow rate inversion is not supported.

If the sum of all the entering mass flow rates at the present iteration is zero, the sum at the previous iteration is taken instead.

MODULE MITN CARD F

Residuals

 

TCALC = 0.

DO I = 1, NE

IF (SUME .GE. WEPS) THEN

AK (I) = AMAX1 (WE(I) / SUME, 0.)

ELSE

AK (I) = DATI (IPD+I)

ENDIF

TCALC = TCALC + TE(I) * AK(I)

ENDDO

C

RNI(1) = WNODO - SUME

RNI(2) = TNODO - TCALC
