Xaing activation

The selection of the Programs -> Xaing button or, more quickly, the selection of the I button, on the Master Menu interface causes the interface activation.

Interface description

The interface is divided into 4 main zones:

 

the menubar

the filter area

the variables list area

the command area

 

The commands provided by the menubar are:

 

File

Quit

to terminate the interface

 

Filter

All variables

to keep variables of all the models

 

Filter

models & blocks

to provide filtering by models & blocks

 

Filter

models only

to provide filtering by model

Filter

KKS filter

to provide filtering by KKS tag

 

Filter

by type

to provide filtering by type of variable (temperature, pressure, etc)

 

 

 

 

 

 

 

The filter area presents the filters currently selected.

 

The variable list area lists all the variables that currently satisfy the filter selection criterion. Furthermore, a "find" function is available for string search.

 

The command area presents two buttons:

 

SELECT to actuate the selected variable

QUIT to terminate this dialog

 

 

Interface operation

Variable selection

The interface is activated with the KKS and the By Type filters activated by default; the By Type filter is locked on the Not Connected Input selection. In that way all the not connected input variables of the simulator that have been tagged with a KKS label are displayed. Further selection among the displayed variables is possible by typing characters in the KKS FILTERS fields and clicking on the APPLY button.

 

Using the menubar, the instructor can activate the Models & Blocks filter or the Models Only filter (they are both alternatives to the All Variables default filter).

In the first case the lists of the models constituting the simulator is displayed along with, for each model, the list of the respective blocks. At the bottom of the Blocks list a "find" function is provided for string search.

In the latter case it is displayed the list of the models only.

 

The By Type filter, always active, makes available the part of the interface that allows to restrict the visibility to the variables that match the type selection (no type selected means no filter applied). The APPLY button starts the procedure that updates the variables list area.

 

The selection is made at the variables list area, eventually using the provided "find" function, clicking on the desired item of the list.

The SELECT button activate the Xaing program for the selected variable.

 

 

Exit the dialog

To close the interface select one of the QUIT buttons provided by the interface.
