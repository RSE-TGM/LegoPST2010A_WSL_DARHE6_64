Instructor Station

 

 

This document is the User Manual of the Instructor Station Interface. It describes first of all the user environment, the configuration files and the file managed

by the instructor station operations (snapshot management, backtrack management, etc.) and by the instructor station utilities (graphics, etc.).

It contains information about the states of the simulator and a general description about the instructor station capabilities.

The core of this document is the description of the instructor station interfaces (description of the functionalities available in the instructor station

and a guide to use them in a typical session).

In Appendix there are examples of user environment configuration, graphical resources configuration, MMI activation and manual simulator shutdown.
