The item selection area shows an option menu that permits to select the item for customization.

Init options

The toggle buttons presented are:

 

Clear

to configure or remove the Clear button from the Master Menu

 

 

 

 

 

Using this option the instructor can avoid the lost of recorded data (the clear function zeroes the simulation time and clear all the recording files).

 

 

Snapshot options

The toggle buttons presented are:

 

Active

to configure or remove the Snapshot button from the Master Menu

 

Cut & Paste

to configure or remove the Cut and Paste buttons from the Snapshot interface

 

Copy & Paste

to configure or remove the Copy and Paste buttons from the Snapshot interface

 

 

 

 

 

 

Through this option it is possible to inhibit the deletion and the rearrangement of snapped I.C..

The Snapshot entry on the Master Menu manubar can also be removed (use the S button, rather).

 

 

 

Protection options

This option is available for the superuser only.

The interface allows to choose, among all the I.C., the ones that will be protected against destruction by non superuser instructor.

The I.C. from 1 to 10, the 59 and 60 are hard protected by the software.

Recording options

The toggle buttons presented are:

 

Active

to configure or remove the Recording button from the Master Menu

 

Backtrack

to configure or remove the Backtrack button from the Master Menu

 

Continuos Recording

to configure or remove the Continuos Recording button from the Master Menu

 

 

 

 

 

The Continuos Recording entry on the Master Menu manubar can be removed; this prevents the possibility to modify the list of the recorded variables while leaving the possibility to activate the Graphics program (G button).

The Backtrack entry on the Master Menu manubar can also be removed (use the B button, instead).

 

 

Display options

The interface allows to enter up to 5 available displays for output the Graphics program. The configured displays are selectable at the time the Graphics program is started.

 

 

User programs option

The interface allows to enter up to 8 UNIX shell commands. The configured commands are activable using the Programs option in the Master Menu.

Attention must be paid to the environment the commands are to be executed: if a configured command executes for a long time (a program that executes compilation, a program that opens graphical interfaces, etc), is to be executed in the background environment. On this purpose terminate the command with the "&" character.

 

 

Application of the current options

To apply the current options, click on the APPLY button when the options selection area displays the modified options. The modifications that apply to the Master Menu interface are immediately visible.

Modifications do not apply to an already opened interface: to activate the modification first close the interface, then call it again.

 

 

Saving the current options

To save the configuration select the File -> Save button.

The modified configuration is saved into the file read by the instructor station at startup.

 

 

Loading the default configuration

To load the default configuration file select the File -> Load button.

The interface closes, and the default values are restored.

 

 

Exit the dialog

To close the interface select one of the QUIT buttons provided by the interface.
