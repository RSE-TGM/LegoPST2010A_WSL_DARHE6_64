Continuos recording

The continuos recording is the automatic function that records the values of a instructor defined set of variables. The recorded values are displayed by the program Graphics in trend format.

The functions that the instructor station provides are basically two:

the possibility to maintain the list of the recorded variables

the possibility to activate the trend program

Activation of the trend program

The trend program (Graphics) can be activated selecting the Recording -> View Recorded Data button or, more quickly, selecting the G button on the Master Menu interface.

A dialog is initiated that allows to select the destination display for the program. Choose the Continue pushbutton to go ahead or the Quit pushbutton to abort the operation.

Using the interface of the trend program the instructor can select and group the recorded variables, modify the engineering units and the limits.

 

 

Filter for configuring the recorded variables

The interface is activated selecting the Recording -> Select Variables button from the master menu interface; it allows the selection of variables among the entire database.

Filter selection

The interface is activated with the KKS filter activated by default; in that way all the variables of the simulator that have been tagged with a KKS label are displayed. Further selection among the displayed variables is possible by typing characters in the KKS FILTERS fields and clicking on the APPLY button.

 

Using the menubar, the instructor can activate the Models & Blocks filter or the Models Only filter (they are both alternatives to the All Variables default filter).

In the first case the lists of the models constituting the simulator is displayed along with, for each model, the list of the respective blocks. At the bottom of the Blocks list a "find" function is provided for string search.

In the latter case it is displayed the list of the models only.

 

The activation of the By Type filter makes available the part of the interface that allows to restrict the visibility to the variables that match the type selection (no type selected means no filter applied); further selection is available filtering by input, output and not connected input variables. The APPLY button starts the procedure that updates the variables list area.

 

 

Adding variables

The selectable variables list area lists all the variables that satisfy the filters criterion and that are not selected for recordings.

New variables can be added to the list of the recorded ones by selecting them from the list.

 

It is possible to select items in the list in several ways:

 

with a single selection, selecting the desired variable using the mouse

with a multiple selection using the mouse, clicking on the first of a contiguous block of variables and dragging the mouse up to the last one of the block

with a multiple selection using the mouse and the keyboard; the use of Ctrl key allows for selection of non contiguous blocks

using the All button, that selects all the variables matching the "find" field, if not empty, or all the variables in the list if empty

 

The Add button activates the function that updates the displayed lists.

 

 

Removing variables

The selected variables list area lists all the variables that satisfy the filters criterion and that are selected for recordings.

Entries can be eliminated from the list of the recorded variables by selecting them from the list.

 

It is possible to select items in the list in several ways:

 

with a single selection, selecting the desired variable using the mouse

 

with a multiple selection using the mouse, clicking on the first of a contiguous block of

variables and dragging the mouse up to the last one of the block

 

with a multiple selection using the mouse and the keyboard; the use of Ctrl key allows for selection of non contiguous blocks

 

using the All button, that selects all the variables matching the "find" field, if not empty, or all the variables in the list if empty

 

The Remove button activates the function that updates the displayed lists.

 

 

Saving and loading sets of variables

 

Once the list of the recorded variables has been configured, it can be saved for future use in an instructor defined file clicking the File -> Save ... button. An interface is opened that provides the possibility to choose the destination file.

 

A previously saved file can be loaded clicking the File -> Load ... button. An interface is opened that provides the possibility to choose the source file.

 

 

Replacing the current set

To immediately activate the selections made by the interface the instructor is required to select the LOAD button from the command area. This request activate a procedure that:

 

updates the files recorder.edf/rtf (the ones that the continuos recording function looks for when initializing)

reinit the program that actually records the data with the new set of variables (a maximum of 1000 variables is permitted)

rearrange the f22circ.hd and f22circ.dat files for the new configuration

 

This procedure take time depending on the differences between the original and the final set of data.
