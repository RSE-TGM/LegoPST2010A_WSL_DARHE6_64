Interface description

 

 

The interface shows, for each model, the name and the milliseconds spent during the last simulation cycle. A bar graph allows graphical comparison among tasks.

 

 

Interface operation

The interface can be opened selecting the Show -> Statistic button on the Master Menu interface. The displayed values are continuously refreshed.

 

 

Exit the dialog

 

To close the interface select one of the QUIT buttons provided by the interface.
