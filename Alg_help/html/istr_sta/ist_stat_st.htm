States of the simulator

 

 

 

Stop

This is the transitory state indicating that the simulator is initializing.

Freeze

In this state the simulator tasks are active but not running. The simulation time is not incremented and the automatic functions of the Instructor Station are not executed.

It is possible to switch to a different state (but REPLAY).

Run

The simulator tasks are running and the Instructor Station executes the automatic functions: backtrack snapshots, continuos recording, commands recording.

Backtrack

The simulator tasks are not running and the Instructor Station does not execute the automatic functions.

From BACKTRACK state it is possible to load the snapshots automatically recorded or initialize the replay function.

Replay

In this state, reached from the BACKTRACK state, it’s possible to "replay" one of the recorded sessions or part of that. The models run like in the RUN state but the inputs coming from the operator are inhibited. Starting from the loaded backtrack snapshot, all of the actions previously recorded (of both the instructor and the operator) are proposed to the simulator at the appropriate time.
