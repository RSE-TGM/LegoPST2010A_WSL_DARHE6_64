LEGOMMI layout: mmi's synoptic page

A mmi's synoptic page displays a drawing area featuring the object configured through config and four push button on the left which activate different exclusive options.

Push button NOP (normal operation)

By pressing NOP, the user enables only the specific call-back of every object (i.e. buttons).

Push button AING

By pressing AING, it is possible to introduce any perturbation in the free input variables of every icon. In fact, by clicking the first mouse button on an active object, the Variabile Editor will be displayed loaded with the Model, Block field, and first animation variable (if free input type var) of the icon already set. The user, can modify the selected variable be means of the list push button of the variable field. Once OK push button has been pressed a predefined perturbation window, called MmiAingWindow will be displayed.

Push button PLOT

By pressing PLOT, the user can display a pop-up run time trend on a variable. Clicking the first mouse button on an active object Variable Editor will be displayed loaded with the Model, Block field, and first animation variable of the icon already set. The user, can modify the selected variable be means of the list push button of the variable field.

Once the OK push button has been pressed, a window featuring a graphic object (Run time one variable type) will be displayed.

Push button STAZ

By pressing STAZ, mmi allows the rising of a station page (defined inside the Geometric resource) associated with an icon. This operation is made by clicking on the icon with the first mouse button.

Push button INFO

By pressing INFO and then clicking on an icon (first mouse button), a window containing a scrolled window featuring the most important resource will be displayed.

Push button CLOSE

By pressing CLOSE, mmi closes the page after displaying a confirm dialog box.

LEGOMMI layout: mmi's station page

A mmi's station page is similar to a synoptic page but has a different layout, in fact it has not the five buttons on the left of the drawing area.

If a station page is connected with an object of a synoptic page (through the dedicated resource) it can be opened by clicking on the object as it is explained on MmiSynopticPage.

LEGOMMI layout: aing window

Through the AING interface, the user can send a perturbation to the selected variable. The window displays the variable name, its value, the measure unit (--- if not defined) and a number of text fields and push buttons necessary to send and start perturbation concerning chosen by selecting toggle button.
