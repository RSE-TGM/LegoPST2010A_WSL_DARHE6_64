
LegoPc release 5.0
========================================================================

*** From release 4.0, the LegoPc system includes some important new features:


   - a new and more easy-to-use interface for modules instantiation. Now it is also 
     possible to modify previous instantiations without starting from scratch

   - a new interface for batch transient perturbation setting

   - full compatibility with:
			Windows NT4 (service pack 5 and following)
			Windows 2000
			Windows XP

   - usage of the last and best performing versions of the packages Tcl-Tk and Tix


*** Release 4.1 fixes some bugs and adds a post-processing utility for steady state


Please, report any usage problems and bugs to: @cesi.it

Before using LegoSim, go to the directory: LegoPc\bin\LegoSim\ReadMe.


*** Release 5.0 include many important features

   - many bugs fixed 

   - a new directory structure of LegoPC that separates the code files and the user files

   - all graph files generated by LegoPC can be read by graphics tool with no translation

   - LgSincro can be run from Lg_Shell

*** Release 5.1 include the following features

   - bugs fixed
   - Steady state can be started inside the modified data activity
   - GUI of lego_wgrs activity modified
   - Lgsincro 1.4: graphic selection of the trends 

Enjoy ...
