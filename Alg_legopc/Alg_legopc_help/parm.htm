MODULE PARM

CARD A

Model Generalities

BAR

 

Module PARM simulates a HV bus bar to be connected to various electrical networks and loads (fig.1).

"Loads" can be motors, generators, lines, loads; that is, everything that can be connected to an electrical network.

A tap changer can be connected to module PARM too.

MODULE PARM

CARD C

Physical and Geometrical Data

The module data are the following:

[V]

VBASE

basic value of bus voltages

[-]

NC

number of connected loads

[-]

NR

number of connected networks

[-]

PU

p.u./absolute values option

NC and NR are integer variables. It must be 0 £ NC £ 9, and 1 £ NR £ 9.

MODULE PARM

CARD F 

Residuals

The equations of the mathematical model are translated in the following residuals:

RNI(1) = -SOMIRR+SOMIRC

RNI(2) = -SOMIXR+SOMIXC

IF (NR.GT.1) THEN

DO J11=2,NR

RNI(2*J11-1) = VRR(1)-VRR(J11)

RNI(2*J11) = VXR(1)-VXR(J11)

ENDDO

ENDIF

RNI(2*NR+1) = VR00-VRR(1)

RNI(2*NR+2) = VX00-VXR(1)

during the steady-state calculation,

RNI(2*NR+3) = -VM00+SQRT(VRR(1)**2+VXR(1)**2)

and for transient simulations,

RNI(2*NR+3) = SQRT(VRR(1)**2+VXR(1)**2)

where SOMIRC and SOMIRR are the total current real components of the loads and of the networks respectively; likewise, SOMIXC and SOMIXR are the imaginary ones.
