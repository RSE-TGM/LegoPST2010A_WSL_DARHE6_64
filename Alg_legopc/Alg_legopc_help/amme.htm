MODULE AMME

CARD A

Model Generalities

VARIABLE ADMITTANCE

The AMME module simulates the beahviour of a passive load, consisting of a variable admittance.

The equations of the AMME module are written according to generator set convention, that is considering currents output as positive, and absorbed currents as negative ones.

Equations used are:

1) Ohm law

2) Current module equation

3) Equations of powers

CARD C

Physical and Geometrical Data

In addition to the initial values of input and output variables of the AMME module, whose meaning has been defined in card B1, the user must assign the following set of data to complete the input required by the module:

[V] VBASE basic voltage between lines of the system in p.u.
[W] QBASE basic power of the system in p.u.
[A] INOM load rated current
[V] VNOM load rated voltage between lines
[W] QNOM load active rated power
[rad/s] OMBA basic system pulsation in p.u.

CARD F 

Residuals

The equations of the mathematical model, mentioned in card A1, originate the following residuals:

RNI(1) = IROO-VXCA*BMOD+VRCA*GMOD

RNI(2) = IXOO+VRCA*BMOD+VXCA*GMOD

RNI(3) = IM00-SQRT(IR00**2+IX00**2)

RNI(4) = QATT-VRCA*IR00-VXCA*IX00

RNI(5) = QREA-VXCA*IR00-VRCA*IX00

RNI(6) = VMCA-SQRT(3.*(VRCA**2+VXCA**2))

RNI(7) = VR00-VRCA

RNI(8) = VX00-VXCA

 

These residuals are valid in switch closed conditions. When the switch is open, residuals are:

RNI(1) = -IROO

RNI(2) = -IXOO

RNI(3) = -IM00

RNI(4) = -QATT

RNI(5) = -QREA

RNI(6) = -VMCA

RNI(7) = -VRCA

RNI(8) = -VXCA

where

    BMOD = load susceptance

    GMOD = load conductance

 

In particular, for load susceptance we have the following expressions:

    BMOD = FB00*BNOM/OMAI for inductive loads

    BMOD = FB00*BNOM*OMAI for capacitive loads

 

where

    BNOM = load rated susceptance
