Adding new ports

PROCEDURE TO DEFINE PORTS SETTINGS

 

The directory containing sources Tcl-Tk-Tix (p.es. c:\LegoPC\Lavoro\Bin\Tix) includes also the file connect.dat.

This file allows redefining the settings of existing connectors and defining new ones.

The file connect.dat uses the following syntax:

 

term Termic

LineColor red

LineWidth 4

LineBitmap None

00tempEsterna

01tempMedia

02coeffScambio

03potenza

****

flue FlueGas

LineColor black

LineWidth 4

LineBitmap None

00portata

01pressione

02temperatura

03tempInvers

04frMolO2

05frMolCO2

06frMolH2O

07frMasH2O

****

****

Each connector type is defined by a certain number of lines. Sections regarding different types of connectors are spearated by a line with four stars: "****". Two such lines conclude the file.

Each connector is defined by:

First line defining a short name (four characters) and a descriptor (free length);
Second line with the key word LineColor followed by the color of the lines connecting two ports of the current type;
Third line containing the key word LineWidth followed by a number identifying the thickness of the lines connecting two ports of the current type;
Fourth line with the key word LineBitmap followed by Yes (or None) if the line is dashed (or not);
As many lines as the number of variables slots of the connector. Each line is described by a progressive two-digits-number followed (without spaces) by the identifier of the slot (type of the associated variable).

 

A new connector can be defined doing the following operations:

Adding a new section to the file connect.dat
Adding in the directory Pixmaps (e.g. c:\LegoPC\Lavoro\Bin\Pixmaps):
Four files gif (one for each orientation)
Four files xbm (one for each orientation)
Four files ppm (one for each orientation)

 

The files must be given the short name already used in the first definition line inside the file connect.dat, followed by the orientation identifier (n,s,w,e).
