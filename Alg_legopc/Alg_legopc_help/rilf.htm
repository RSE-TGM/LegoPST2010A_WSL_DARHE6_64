MODULE RILF

CARD A

Model Generalities

VOLTAGE REGULATOR

 

The module RILF calculates the value of angular velocity in a node, once provided the angular velocity of the reference generator and the node voltage (real and imaginary parts).

Indeed, the angular velocity in a node (OMEGAn) is equal to the reference one (OMEGAr) plus DELTOM and DELTOM is tied to the derivatives of the real and imaginary parts of node voltage.

If Vm is the module, Vr the real part, Vx the imaginary part of the node voltage:

                                           Vr   dVx    Vx    dVr

     OMEGAn = OMEGAr + DELTOM = OMEGAr + ----- ---- - ----- ----

                                         Vm**2  dt    Vm**2  dt

 

Approximating the derivatives with incremental ratios:

                                            Vr   DVx    Vx    DVr

     OMEGAn = OMEGAr + DELTOM =~ OMEGAr + ----- ---- - ----- ----

                                          Vm**2  Dt    Vm**2  Dt

 

If the node tension is zero, also the angular velocity is zero.

In steady state DELTOM = 0 and OMEGAn = OMEGAr

MODULE RILF

CARD F 

Residuals

 

Steady state:

    RNI(1) = OMNO-OMGE

 

In transients:

    If node voltage is negligible:

        RNI(1) = OMNO

    Else:

    RNI(1)=OMNO*OMBA-OMGE*OMBA-(-UX00*(UR00-UR00P)+UR00*(UX00-UX00P))/M1

    where

    M1=(UR00**2.+UX00**2.)*DTINT
