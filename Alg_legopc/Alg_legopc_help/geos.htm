MODULE GEOS

CARD A

Model Generalities

OSCILLATING VALUE GENERATOR

 

In some applications, the user needs an input variable with an oscillating value, without reproducing the phenomena that are responsible of this oscillation. The module GEOS allows to obtain this variable fluctuation, in transient state. The fundamental characteristics of module are:

Average value and oscillating amplitude as input variables

Time to time oscillating value as output variable

Pulsatory speed as data input.

Oscillation of sinusoidal type

Analitic jacobian

In steady state, output equal to average value as taken from input

CARD C

Physical and Geometrical Data

In addition to the variables listed card B, the user must assign the following physical and geometrical DATA:

[rad/s] PULSAZ Pulsatory speed for output value

MODULE GEOS

CARD D

Module Variables Initialization

No particular prescription is required for the module initialization: in steady state the average value will be equal to output instantaneous value. The same will be in transient state, at time equal zero, however the residual should be able to give that condition.

CARD E 

User's Subprograms

MODULE GEOS

CARD F 

Residuals

The expression of the unique residual is:

RNI(1) = YMED-YUSC+AMPL*SIN(OMEG*TEMPO)

 

where:

YMED, AMPL are the same input variables

YUSC is the same output variable

OMEG is the same of datum PULSAZ

the reader can see that in steady state, being TEMPO = 0, YMED = YUSC
