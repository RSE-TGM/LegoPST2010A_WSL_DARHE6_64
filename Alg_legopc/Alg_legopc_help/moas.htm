MODULE MOAS

CARD A

Model Generalities

ASYNCHRONOUS MOTOR

 

The MOAS module simulates the operation of an asynchronous electric motor.

It is typically used together with the CPOM module (centrifugal pump) to implement a model of pump complete with electric motor.

The model consists of a single equation calculating the mechanical power supplied by an asynchronous electric motor according to the rotor angular speed, the applied voltage, the supply frequency and the characteristic power-slipping curve.

CARD B

Topological data

INPUT - OUTPUT SCHEME

 

 

 

PHYSICAL MEANING OF MODULE INPUT-OUTPUT VARIABLES

[rad/s] OMEG angular speed of motor and of load applied to the same (e.g. a pump)
[p.u.] YTEN ratio between the supply voltage applied and the supply rated voltage
[rpm] YFRQ motor supply frequency
[p.u.] YSMO slip value corresponding to maximum power
[W] QMOT  power supplied by motor

MODULE MOAS

CARD C

Physical and Geometrical Data

Meaning of data requested by the input of MOAS module:

[ ] NPOL number of motor pole pairs
[W] PMAX maximum power supplied by motor

The maximum power supplied is the peak value in the absorbed power-slip curve usually supplied by the manufacturer.

MODULE MOAS

CARD F 

Residuals

 

The module includes a unique residual, mentioned hereunder, relevant to the calculation of the mechanical power supplied by the motor.

RNI(1) = QMOT-2.*PMAX*YTEN*YTEN*SCOR/(YSMO+SCOR*SCOR/YSMO)/H0/W0

SCOR is the motor slip while the remaining variables have already been defined in cards B and C.

The residual is normalized with respect to power W0*H0.
