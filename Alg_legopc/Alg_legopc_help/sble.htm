MODULE SBLE

CARD A

Model Generalities

TRANSMISSION NETWORK BUS

 

The module SBLE represents a transmission network bus, which can connect two lines (module LINE) and feed a load.

Input variable SBIS denotes if the bus is feeded by a line (or by any supply), or not.

In the second case (SBIS>0,5), voltage is setted to zero.

If the bus is feeded (SBIS<0,5), the equations of the node currents Kirchhoff's balance are given.

MODULE SBLE

CARD B

Topological data

INPUT-OUTPUT SCHEME

 

 

PHYSICAL MEANING OF THE MODULE INPUT OUTPUT VARIABLES

[V] V1RE bus real voltage
[V] V1IM bus imaginary voltage
[A] I1RE line 1 current real component
[A] I1IM line 1 current imaginary component
[A] I2RE line 2 current real component
[A] I2IM line 2 current imaginary component
[A] ICRE load current real component
[A] ICIM load current imaginary component
[-] SBIS isolated-bus logical-variable

MODULE SBLE

CARD F 

Residuals

The equations of the mathematical model are translated in the following residuals:

If the bus is feeded (SBIS<0,5)

RNI(1) = V1RE

RNI(2) = V1IM

otherwise

RNI(1) = I1RE+I2RE+ICRE

RNI(2) = I1IM+I2IM+ICIM
