MODULE DISN

CARD A

Model Generalities

LINEAR ALGEBRAIC EQUATIONS

 

In the DISN module, differently from most of LEGOCAD library modules, the coded equations do not represent the physical-mathematical model of a given plant component, but a set of linear algebraical relations (max 30) of the following kind:

Xu(j) = K * Xi (j = 1, 2, ..., n)

Once the K coefficient and the Xi variable values have been assigned, the module determines the Xu(j) variables values.

During the topology construction activity, the module requires an alphanumeric character that will be used as the initial character of the FORTRAN symbol adopted for the Xu(j) output variables.

In the following paragraph the function of such a character is carried out by the letter "U".

n = number of output variables (max 30)

[ ] AING known variable

MODULE DISN

CARD C

Physical and Geometrical Data

In addition to initial values of DISN module input and output variables (meanings defined above), user must assign the following data set to complete the module input on the file F14:

[ ]

NORM 

alphanumeric value identifying the normalization factor of the input variable (also used to normalize the output variables) according to the following rule:
blank = 1.
W0 = default value for flow rates
H0 = default value for enthalpies
T0 = default value for temperatures
P0 = default value for pressures
Q0 = default value for powers (= W0 * H0)
RO0 = default value for densities
AL0 = default value for lengths
V0 = default value for volumes
DP0 = default value for pressure differences
numeric value = value to be used as normalization factor

 

[ ]

COEF

input variable coefficient.

MODULE DISN

CARD F 

Residuals

 

The linear algebraical equation, quoted in first paragraph, gives rise to the following residuals:

RNI(j) = U_j_ - COEF * AING (j = 1, 2,..., n)

(normalization factor = NORM)

where U_j_ are the output variables to be calculated, AING is the known input variable, COEF and NORM are defined above.
