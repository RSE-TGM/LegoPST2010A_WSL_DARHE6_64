MODULE SB4N

CARD A

Model Generalities

FOUR-BUSSES PARALLEL

 

The module SB4N simulates the operation of a four-bus bar. Each bus of the bar can be connected to each other through the breakers A, B, C and D.

This module is implemented to represent the LV bus bars feeding the auxiliary services of a typical italian power station.

[A] IX1j jth load current imaginary component from bus 2

MODULE SB4N

CARD F 

Residuals

 

During steady-state calculation,

RNI(9) = -VM01+SQRT(VR01*VR01+VX01+VX01)

RNI(10) = -VM02+SQRT(VR02*VR02+VX02+VX02)

RNI(11) = -VM03+SQRT(VR03*VR03+VX03+VX03)

RNI(12) = -VM04+SQRT(VR04*VR04+VX04+VX04)

during transient simulations,

RNI(9) = SQRT(VR01*VR01+VX01+VX01)

RNI(10) = SQRT(VR02*VR02+VX02+VX02)

RNI(11) = SQRT(VR03*VR03+VX03+VX03)

RNI(12) = SQRT(VR04*VR04+VX04+VX04)

 

If all the bus bars are in parallel (i.e., A, B, C and D breakers closed)

RNI(1) = IR1C-SOMIR01+IR2C-SOMIR02+IR3C-SOMIR03+IR4C-SOMIR04

RNI(2) = IX1C-SOMIX01+IX2C-SOMIX02+IX3C-SOMIX03+IX4C-SOMIX04

RNI(3) = VR01-VR02

RNI(4) = VX01-VX02

RNI(5) = VR01-VR03

RNI(6) = VX01-VX03

RNI(7) = VR01-VR04

RNI(8) = VX01-VX04

 

If A, B, C and D open

RNI(1) = IR1C-SOMIR01

RNI(2) = IX1C-SOMIX01

RNI(3) = IR2C-SOMIR02

RNI(4) = IX2C-SOMIX02

RNI(5) = IR3C-SOMIR03

RNI(6) = IX3C-SOMIX03

RNI(7) = IR4C-SOMIR04

RNI(8) = IX4C-SOMIX04

 

If A, B and C closed, D open

RNI(1) = IR1C-SOMIR01+IR2C-SOMIR02+IR3C-SOMIR03

RNI(2) = IX1C-SOMIX01+IX2C-SOMIX02+IX3C-SOMIX03

RNI(3) = VR01-VR02

RNI(4) = VX01-VX02

RNI(5) = VR01-VR03

RNI(6) = VX01-VX03

RNI(7) = IR4C-SOMIR04

RNI(8) = IX4C-SOMIX04

 

If A and B closed, C and D open

RNI(1) = IR1C-SOMIR01+IR2C-SOMIR02

RNI(2) = IX1C-SOMIX01+IX2C-SOMIX02

RNI(3) = VR01-VR02

RNI(4) = VX01-VX02

RNI(5) = IR3C-SOMIR03

RNI(6) = IX3C-SOMIX03

RNI(7) = IR4C-SOMIR04

RNI(8) = IX4C-SOMIX04

The other possible cases are treated in a similar manner.

SOMIR01 is the real component of the loads total current at bus 1,

SOMIX01 is the imaginary one,

SOMIR02 is the real component of the loads total current at bus 2.

and so on.
