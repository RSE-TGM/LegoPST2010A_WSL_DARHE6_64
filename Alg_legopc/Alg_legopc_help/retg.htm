MODULE RETG

CARD A

Model Generalities

GAS TURBINE REGULATOR

 

The RETG module simulates the operating of a gas turbine regulator.

The regulation (see fig.1) includes the control of the machine rotation velocity and the control of the flue gas temperature inside the combustion chamber. The latter is performed by modulating both the Inlet Guide Vanes (IGV) and the fuel flow rate.

The velocity control is proportional while the temperature control is proportional-integral.

fig. 1. Regulation block diagram

 

The RETG module must always be used together with the TURG module from which it receives as input variables the machine rotation velocity, the flue gas temperature measured by the thermocouples placed at the turbine outlet and the temperature set-point to control the flue gas temperature inside the combustion chamber.

The fuel request and the IGV position resulting from the regulation are furnished as output variables by RETG module.

During the gas turbine start-up, trip and shut-down procedures, pointed out through proper variables, the regulation scheme outlined in fig. 1 is ignored, and the module just assignes to the output variables ZIGV and ZWCO, respectively, the values of the variables ZMIG and ZWMC provided in input by TURG module.

INPUT-OUTPUT SCHEME

 

 

 

PHYSICAL MEANING OF MODULE INPUT-OUTPUT VARIABLES

[K] TTER Flue gas temperature measured by the thermocouples
[K] TTRX Temperature set-point to control the flue gas temperature inside the combustion chamber
[rpm] RPM Gas turbine set rotation velocity (mains frequency)
[p.u.] YSET Revolution variation gear position
[p.u.] YRPM Reference frequency value
[%] ZWMC Fuel flow rate demand (see note below)
[%] ZMIG Inlet Guide Vanes (IGV) position (see note below)
[ ] ALAV Start-up push button

(for ALAV = 1., 2. no regulation is applied and the following assignments are made:

ZIGV = ZMIG

ZWCO = ZWMC

for different from 1. and 2. ALAV values the scheme illustrated in fig. 1 is carried out)
[ ] ALSC Trip push button

(for ALSC = 1. no regulation is applied and the following assignments are made:

ZIGV = ZMIG

ZWCO = ZWMC

for different from 1. ALSC values the scheme illustrated in fig. 1 is carried out)
[ ] ALSH  Shut-down push button

(for ALSH = 1. no regulation is applied and the following assignments are made:

ZIGV = ZMIG

ZWCO = ZWMC

for different from 1. ALSH values the scheme illustrated in fig. 1 is carried out)
[%] ZIGV Inlet Guide Vanes (IGV) position
[%] ZWCO

Fuel flow rate demand
[%] FSRT Requested fuel flow rate in temperature clipping
[%] FSRV Set fuel flow rate and frequency error

Note

During the start-up, trip and shut-down transients, the values for ZWMC and ZMIG, needed to simulate such manoeuvres, are supplied by the functions generators (or tables) implemented in the TURG module. RETG module just acquires these values as input variables and assigns them to the corresponding output variables ZWCO and ZIGV, respectively. In this way the functions generators feedback on the TURG module input variables, necessary for the simulation of the aforesaid manoeuvres, is performed.

MODULE RETG

CARD C

Physical and Geometrical Data

In addition to initial values of RETG module input and output variables (meanings defined above), user must assign the following data set to complete the module input on the file F14:

[s] TAU1 Time constant of the IGV position control feedback loop
[%/K] VTPG IGV closing percent per degree of temperature difference between the flue gas temperature and the reference one
[%] VINF IGV closing minimum value in full speed - no load condition
[s] TAU2 Time constant of the temperature control feedback loop
[%] STAT Permanent speed droop
[%/K] GAIN Proportional gain of the temperature control
[K] DESE Desensitization value of the temperature control
[rpm] RPMX  Rotation velocity limit value

(if RPM > RPMX the fuel demand WCM present in the medium value selector of fig.1 takes the value -5.

if RPM <= RPMX the fuel demand WCM present in the medium value selector of fig.1 takes the value 0.)
[s] RTERM Thermocouple time constant
[s] RPOGV IGV positioner time constant

Residuals

 

The equations forming the RETG module mathematical model give rise to a set of residuals. Here only their correspondences with the output variables equations have been indicated, also mentioning the respective normalization factors. Further explanations are given in the manual section: "LEGOCAD - Modelling Library for Simulation of Power Generation Plants - Theory - Modules Description".

1) RNI(1) - normalization factor = 100.

related to the Inlet Guide Vanes (IGV) position

2) RNI(2) - normalization factor = 100.

related to the fuel flow rate
