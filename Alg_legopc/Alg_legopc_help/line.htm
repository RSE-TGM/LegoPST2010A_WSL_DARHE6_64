Physical and Geometrical Data

The data of the module are:

[W /Km] RES series resistance per unit length
[W /Km] XL series reactance per unit length
[S/Km] BMEZZI shunt half susceptance per unit length
[S/Km] GMEZZI shunt half conductance per unit length
[Km] LUNG line length
[V] VNOM rated voltage
[-] PU p.u./absolute values option (1à in p.u.; 0à absolute)

Residuals

The equations of the mathematical model are translated in the following residuals:

RNI(5) = FREQ-FROU

 

If the breakers A and B are closed,

RNI(1) = I1RE+(Y11RE*V1RE)-(Y11IM*V1IM)+ (Y12RE*V2RE)-(Y21IM*V2IM)

RNI(2) = I1IM+(Y11RE*V1IM)+(Y11IM*V1RE)+ (Y12IM*V2RE)+(Y12RE*V2IM)

RNI(3) = I2RE+(Y21RE*V1RE)-(Y21IM*V1IM)+(Y22RE*V2RE)-(Y22IM*V2IM)

RNI(4) = I2IM+(Y21IM*V1RE)+(Y21RE*V1IM)+(Y22RE*V2IM)+(Y22IM*V2RE)

 

If A is closed, and B is open

RNI(1) = I1RE+(YEQRE*V1RE)-(YEQIM*V1IM)

RNI(2) = I1IM+(YEQRE*V1IM)+(YEQIM*V1RE)

RNI(3) = I2RE

RNI(4) = I2IM

 

If B is closed, and A open

RNI(1) = I1RE

RNI(2) = I1IM

RNI(3) = -I2RE+(YEQRE*V2RE)-(YEQIM*V2IM)

RNI(4) = -I2IM+(YEQRE*V2IM)+(YEQIM*V2RE)

 

If A and B are open

RNI(1) = I1RE

RNI(2) = I1IM

RNI(3) = I2RE

RNI(4) = I2IM

 

Base power:

QBASE = H0*W0

 

Real and imaginary components of the series and shunt impedances, in per unit of base impedance Zn=(VNOM)2/QBASE:

ZMRE = RES*LUNG*QBASE/(VNOM*VNOM)

ZMIM = XL*(FREQ/50.)*LUNG*QBASE/(VNOM*VNOM)

ZARE = ((GMEZZI/BMEZZI**2)/LUNG)*((50./FREQ)**2)* *QBASE/(VNOM*VNOM)

ZAIM = -(1/BMEZZI*LUNG))*(50./FREQ) * *QBASE/(VNOM*VNOM)

COMPLEX variables are defined

ZA

= CMPLX(ZARE,ZAIM)

(ZA = R+j·XL)

ZM

= CMPLX(ZMRE,ZMIM)

(ZM = (G/2+j·B/2)-1)

Admittances matrix elements

Y11

= (ZA+ZM)/(ZA*ZM)

Y12

= -1/ZM

Y21

= -Y12

Y22

= -Y11

YEQ = (2*ZA+ZM)/((ZA+ZM)*ZA)

= REAL(Y11)

Y11IM

= AIMAG(Y11)
