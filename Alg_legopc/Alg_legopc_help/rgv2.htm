MODULE RGV2

CARD A

Model Generalities

VOLTAGE REGULATOR

 

This module simulates the behaviour of a primary voltage regulator, where under- and overexcitation limites of the synchronous generator are taken into account.

MODULE RGV2

CARD C

Physical and Geometrical Data

The data of the module are:

[s] T1 pole time constant of the transfer function
[s] T2 zero time constant of the transfer function
[-] GAIN transfer function gain
[s] TO time constant of the overexcitation error integrator
[s] TU time constant of the underexcitation error integrator
[A] IFL maximum excitation current
[VAR] QV0 maximum reactive power in underexcitation
[-] BETA inverse of the angolar coefficient of the line defining the underexcitation limits
[V] VNOM generator rated voltage
[V] VBEC basic excitation voltage
[p.u.m.] VC1 maximum excitation voltage
[p.u.m.] VC2 minimum excitation voltage
[W ] RF excitation circuit resistance
[-] PU p.u./absolute values option (1à in p.u.; 0à absolute)

The machine per unit is indicated with p.u.m., that is the one calculated with the basic excitation voltage.

MODULE RGV2

CARD F 

Residuals

The equations of the mathematical model are translated in the following residuals:

During steady-state calculations,

RNI(1) = VOEX

RNI(4) = VOE1

RNI(2) = VUEX

RNI(5) = VUE1

RNI(3) = VF11-GAIN*(1-T2/T1)*(VRIF+VPS0-VV)

RNI(7) = (VFAC-GAIN*(VNOM/VBEC)*(VRIF+VPS0-VV))/GAIN

 

during transient simulations,

IF (IFLD.LT.(IFL/IFLD0)) THEN

RNI(1) = 0.

RNI(4) = VOE1 (reset of the overexcitation limiter signal)

ELSE

RNI(1) = (1/TO)*(IFLD-(IFL/IFLD0))

RNI(4)=VOE1-VOEX

ENDIF

IF ((QREA-((QV0/QBASE)+QATT*BETA)).GT.0.) THEN

RNI(2) = 0.

RNI(5) = VUE1 (reset of the underexcitation limiter signal)

ELSE

RNI(2) = (1/TU)*(QREA-((QV0/QBASE)+QATT*BETA))

RNI(5) = VUE1-VUEX

ENDIF

RNI(3) = -VF11/T1+(GAIN*(1.-T2/T1))*(VRIF+VPS0-VV)/T1

RNI(7) = (VFAC-VF11*(VNOM/VBEC)- GAIN*(VNOM/VBEC)*T2/T1*(VRIF+

+VPS0-VV))/GAIN

 

RNI(6) = VRF0-VOE1-VUE1-VRIF

IF (VFAC.GE.VC1) THEN

RNI(8) = VF00-VC1

ELSE IF (VFAC.LE.VC2) THEN

RNI(8) = VF00-VC2

ELSE

RNI(8) = VF00-VFAC

ENDIF

Where,

QBASE=H0*W0 (basic power)

VV = VMMA

IFLD0=VBEC/RF (basic excitation current)
