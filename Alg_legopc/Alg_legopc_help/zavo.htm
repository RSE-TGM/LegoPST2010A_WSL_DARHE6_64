MODULE ZAVO

CARD A

Model Generalities

"BALLAST" LOAD

 

The ZAVO module simulates the typical behaviour of an aggregate load when reenergized, shown in fig.1 and fig.2 (active and reactive output respectively):

 

By feeding a load, active output experimental profiles show generally a very short peak (P1 is the amplitude, T1 the duration), with a final value (P2), which can be bigger or lower than the rated one (PNOM). This depends of the former deenergization interval duration: if it was short (matter of seconds), P2 appears lower (as in fig.1); otherwise, it shall be bigger.

After that, the load returns to the normal value PNOM in matter of minutes (T2).

An exponential profile has been taken on for describing the reactive output, according to experimental results (fig.2), where T3 lasts a few seconds as well as T1.

Active and reactive output dependence on voltage module is exponential:

where V0 is the load rated voltage, and P0 and Q0 are referred to V=V0.

Other parameters, like P1 and T1, depend of the deenergization time. So it is worth noting that very different behaviours can be expected for the same load area. Then, it is essential to have at disposal exaustive experimental data.

However, user is requested to introduce at least the following data: PU, PNOM, QNOM; concerning the remaining ones, default values are already present.

Notice that peak amplitudes are input variables, not data (then, there are no defaults for them).

MODULE ZAVO

CARD C

Physical and Geometrical Data

The data of the module are:

[V] VNOM rated voltage
[W] PNOM active rated output
[VAR] QNOM reactive rated output
[W] P2 final value of the active output peak
[s] T1 time corresponding to P2
[s] T2 time to active rated output
[s] T3 time to reactive rated output
[ ] KPV exponent of active output dependence on voltage module
[ ] KQV exponent of reactive output dependence on voltage module
[ ] PU p.u./absolute values option (1à in p.u.; 0à absolute)

The default values are the following:

P2 1.2*PNOM
T1 .5
T2 900.
T3 .5
KPV 1.25
KQV 2.5

CARD F 

Residuals

 

The equations of the mathematical model are translated in the following residuals:

If the load is energized,

RNI(1) = IR00+(PASS*VR00+QASS*VX00)/(VR00*VR00+VX00*VX00)

RNI(2) = IX00+(PASS*VX00-QASS*VR00)/(VR00*VR00+VX00*VX00)

    during steady-state calculations (KREGIM =.TRUE.)

RNI(3) = -QATT+VR00*IR00+VX00*IX00

RNI(4) = -QREA+VX00*IR00+VR00*IX00

RNI(5) = -IM00+SQRT(IR00*IR00+IX00*IX00)

RNI(6) = -VM00+SQRT(VR00*VR00+VX00*VX00)

RNI(7) = -FDIP+FDP0,

            where, if complex power output QAPP is not zero,

            FDP0 = -QATT/QAPP

            otherwise

            FDP0 = 0

    in transient simulations (KREGIM = .FALSE.)

RNI(3) = VR00*IR00+VX00*IX00

RNI(4) = VX00*IR00+VR00*IX00

RNI(5) = SQRT(IR00*IR00+IX00*IX00)

RNI(6) = SQRT(VR00*VR00+VX00*VX00)

RNI(7) = FDP0,

where, if QAPP is not zero,

FDP0 = -QATT/QAPP

otherwise

FDP0 = 0

 

If the load is not energized,

 

RNI(1) = IR00

RNI(2) = IX00

    steady-state calculations (KREGIM =.TRUE.)

RNI(3) = -QATT

RNI(4) = -QREA

RNI(5) = -IM00

RNI(6) = -VM00

RNI(7) = -FDIP

    transient simulations (KREGIM = .FALSE.)

RNI(3) = 0

RNI(4) = 0

RNI(5) = 0

RNI(6) = 0

RNI(7) = 0

where:

VV = SQRT(VR00*VR00+VX00*VX00)

QBASE = H0*W0

 

STATO logical variable is .TRUE. if the load breaker is closed, and if supplying voltage module is greater than 65% of the rated one. STATO_PREC variable keeps the previous integration step value of STATO.

Through a comparison between STATO and STATO_PREC, the last temporal step (registered in the TEMPO_CAMBIO variable) in which the load, during a transient, has changed its state (connected or disconnected), is recognized: whenever the load begins to be supplied, the TEMPO_ATTUALE variable is set to zero (TEMPO_ATTUALE = TEMPO-TEMPO_CAMBIO).

Active and reactive output expressions (identified, in the program, in PASS and QASS variables) are assigned in the following way:

Disconnected load (STATO =.FALSE.),

PASS = 0.

QASS = 0.

 

Connected load (STATO =.TRUE.),

IF (TEMPO_ATTUALE.GE.T2) THEN

    PASS = FP00*(PNOM/QBASE)*VV**KPV

ELSE IF

    PASS = FP00*(P2/QBASE)+((PNOM/QBASE)-(P2/QBASE))*(TEMPO_ATTUALE-T1)/(T2-T1))*VV**KPV

ELSE

    = FP00*(P100-(P100-(P2-QBASE))*TEMPO_ATTUALE/T1)*VV**KPV

END IF

IF (TEMPO_ATTUALE.GE.T3) THEN

    QASS = FQ00*(QNOM/QBASE)*VV**KQV

ELSE

    QASS = FQ00*(Q100-(Q100-(QNOM/QBASE))*(1-EXP(-TEMPO_ATTUALE/(T3/3)))/(1-EXP(-3.)))*VV***KQV

END IF
