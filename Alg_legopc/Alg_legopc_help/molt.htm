MODULE MOLT

CARD A

Model Generalities

ANALOG MULTIPLIER OF THREE SIGNALS

 

The module MOLT multiplies three analog signals.

MODULE MOLT

CARD B

Topological data

INPUT-OUTPUT SCHEME

 

 

PHYSICAL MEANING OF MODULE INPUT-OUTPUT VARIABLES

[…] XPRO product
[…] XFA1 first factor
[…] XFA2 second factor
[…]  XFA3 third factor

MODULE MOLT

CARD C

Physical and Geometrical Data

 

In addition to the initial values of input and output variables, the user must assign the following data:

[…] NORUSC product variable normalization
[…] NORIN1 first factor normalization
[…] NORIN2 second factor normalization
[…]  NORIN3 third factor normalization

MODULE MOLT

CARD D

Module Variables Initialization

To initialize the MOLT module variables no particular prescription is required.
