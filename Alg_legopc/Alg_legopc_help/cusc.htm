MODULE CUSC

CARD A

Model Generalities

JOURNAL BEARINGS

 

The CUSC module simulates the behaviour of a group of journal bearings, the number of which can be selected by the user. All of these bearings act in indipendent way, also for their input variables, except for the rotation speed that is unique. All of these bearings have a metallic mass, that absorbs the production of thermal power related to the friction. Each of these bearings can take into account also a vibrational behaviour part: in this case the module calculates the oscillating amplitude and eventually a surplus of heat production related to an eventually settable "bad work state".

The complete bearing, with vibrations computation included, is called "radial bearing", the one with only thermal behaviour "axial bearing".

In any way, each bearing can be easily connected to heat "semiexchanger" (i.e. PORE) or some other module capable of an heat exchange.

Main hypothesis in CUSC module are, for each bearing:

- Heat rate production is proportional to rotation speed

- Heat rate production is homogeneous in metallic mass

 

Equations used to build the model are:

for all bearings:

- equation of conservation of energy

- equation of friction heat production

 

only for radial bearings:

- characteristics curves of vibrations

MODULE CUSC

CARD B

Topological data

INPUT -OUTPUT SCHEME

 

NCUSC = total number of bearings

NRAD = number of radial bearings (0 £ NRAD £ NCUSC)

 

i = 1, NCUSC

j = 1, NRAD

 

Note: ADIj variables are present only if NRAD > 0

 

PHYSICAL MEANING OF MODULE INPUT - OUTPUT VARIABLES

[W] QREi Outgoing power from the i-th bearing
[rad/s] RPMT Rotational speed
[mm] SUOi Imposed anomaly of vibration amplitude
[K] TMAi Metal temperature of i-th bearing
[mm] ADIj Vibration amplitude of j-th radial bearing

MODULE CUSC

CARD C

Physical and Geometrical Data

In addition to the initial values of input and output variables of CUSC module, whose meaning has been defined in CARD B, the user must assign the following set of data:

Generally:

[RPM] RPMNOM Rated rotational speed (default = 3000.)
[J/rad] KROTOLAM Linear coefficient putting into relation heat production [W] and rotational speed (rad/s) (default = 711.)
[1/mm] %POT/AMP Percentage of heat overproduction related to bad working condition (default = 1.) (see Note 1)

All of this data have a default value: putting a negative value, the value between parenthesis is taken.

For each bearings:

[ ] RIPATTi  Fraction of total heat production absorbed by i-th bearing.
WARNING: the user must control that the sum is one; the module doesn’t check it as some applications can require a different sum of fraction.
[kg] MASSAi Mass of i-th bearing
[J/kg/K] CALSPE Specific heat of material of i-th bearing (default = 500.)

Only for radial ones:

[mm] VIBRISj Oscillating amplitude at the critical rotational speed
[RPM] RPMCRIj Critical rotational speed
[mm] VIBREGj Oscillating amplitude at rated rotational speed

With these three informations, for each bearing the oscillating amplitude can be calculated: the curve of its value will intersecate zero for RPM=0, VIBRISi for RPM = RPMCRIi and VIBREGi for RPM = RPMNOM.

Note 1: For example, the value 2. means that, if an oscillating anomaly of 15 micron is imposed on i-th bearing, the heat produced becomes (1 + 2 X 15/100) times greater than the normal amount.

MODULE CUSC

CARD D

Module Variables Initialization

The QREi are input of the module, if they are not connected with a model of cooling system, in steady state the user must impose the temperatures of material. In transient state indeed, if QREi does not equal the heat production, the TMAi will not join a steady temperature.

MODULE CUSC

CARD F 

Residuals

Residuals have the following expression:

RNI(I) = (Y(I) * QROT * (1. + SN * SAMP(I)) + QREF(I)) / CT(I) / T0

(norm. factor T0) (for all the bearings)

RNI(NCUSC + J) = (AMP(J) + SAMP(J) - ADIN(J) ) / A0

(norm. factor A0 = 10000.) (for radial bearings)

 

where

AMP (J) = oscillating amplitude calculated by the curve vibration vs. RPM
SAMP (J) = additional amplitude imposed as input
ADIN (J)  =total amplitude, as read in XYU vector
Y(I) = the same of RIPATTRi, from DATI
SN = the same of %POT/AMP from DATI, divided by 100
QROT = total heat production
QREF (I) = the same of QREi, from variables
CT (I) = thermal capacity of i-th bearing
