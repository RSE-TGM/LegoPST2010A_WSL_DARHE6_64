MODULE SB3N

CARD A

Model Generalities

THREE-BUSSES PARALLEL

 

The module SB3N simulates the operation of a three-bus bar. Each bus of the bar can be connected to each other through the breakers INTA and INTB.

This module is implemented to represent the MV bus bars feeding the auxiliary services of a typical italian power station.

MODULE SB3N

CARD F 

Residuals

The equations of the mathematical model are translated in the following residuals:

During steady-state calculation,

RNI(7) = -VM01+SQRT(VR01*VR01+VX01+VX01)

RNI(8) = -VM02+SQRT(VR02*VR02+VX02+VX02)

RNI(9) = -VM03+SQRT(VR03*VR03+VX03+VX03)

during transient simulations,

RNI(7) = SQRT(VR01*VR01+VX01+VX01)

RNI(8) = SQRT(VR02*VR02+VX02+VX02)

RNI(9) = SQRT(VR03*VR03+VX03+VX03)

 

If all the bus bars are in parallel (i.e., A and B breakers are closed)

RNI(1) = IR1C-SOMIR1+IR2C-SOMIR2+IR3C-SOMIR3

RNI(2) = IX1C-SOMIX1+IX2C-SOMIX2+IX3C-SOMIX3

RNI(3) = VR01-VR02

RNI(4) = VX01-VX02

RNI(5) = VR01-VR03

RNI(6) = VX01-VX03

 

If A and B open

RNI(1) = IR1C-SOMIR1

RNI(2) = IX1C-SOMIX1

RNI(3) = IR2C-SOMIR2

RNI(4) = IX2C-SOMIX2

RNI(5) = IR3C-SOMIR3

RNI(6) = IX3C-SOMIX3

 

If A closed and B open

RNI(1) = IR1C-SOMIR1+IR3C-SOMIR3

RNI(2) = IX1C-SOMIX1+IX3C-SOMIX3

RNI(3) = IR2C-SOMIR2

RNI(4) = IX2C-SOMIX2

RNI(5) = VR01-VR03

RNI(6) = VX01-VX03

 

If A open and B closed

RNI(1) = IR1C-SOMIR1

RNI(2) = IX1C-SOMIX1

RNI(3) = IR2C-SOMIR2+IR3C-SOMIR3

RNI(4) = IX2C-SOMIX2+IX3C-SOMIX3

RNI(5) = VR02-VR03

RNI(6) = VX02-VX03

 

SOMIR1 is the real component of the loads total current at bus 1,

SOMIX1 is the imaginary one,

SOMIR2 is the real component of the loads total current at bus 2.
