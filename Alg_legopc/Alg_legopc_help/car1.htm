MODULE CAR1

CARD A

Model Generalities

CONSTANT ADMITTANCE LOAD

Module CAR1 represents a static load, consisting by a pure admittance Y = FG00 * *GNOM + j*FB00*BNOM. Coefficients FG00 and FB00 have been introduced in order to simulate load variations, by modifying real and imaginary component of admittance Y, respectively.

FG00 and FB00 are input variables.

MODULE CAR1

CARD F 

Residuals

The equations of the mathematical model are translated in the following residuals:

 

If the load breaker is closed (SISC=1)

RNI(1) = IR00+VR00*FG00*GNOM-VX00*FB00*BNOM

RNI(2) = IX00+VX00*FG00*GNOM-VR00*FB00*BNOM

During steady-state calculations,

RNI(3) = -QATT+VR00*IR00+VX00*IX00

RNI(4) = -QREA+VX00*IR00+VR00*IX00

RNI(5) = -IM00+SQRT(IR00*IR00+IX00*IX00)

RNI(6) = -VM00+SQRT(VR00*VR00+VX00*VX00)

RNI(7) = -FDIP+FDP0

During transient simulations,

RNI(3) = VR00*IR00+VX00*IX00

RNI(4) = VX00*IR00+VR00*IX00

RNI(5) = SQRT(IR00*IR00+IX00*IX00)

RNI(6) = SQRT(VR00*VR00+VX00*VX00)

IF (QAPP.GT.0.00001) THEN

   FDP0 = -ABS(QATT)/QAPP

ELSE

   FDP0 = 0.

END IF

RNI(7) = FDP0

 

If the load breaker is open (SISC=0),

RNI(1) = IR00

RNI(2) = IX00

During steady-state calculations,

RNI(3) = -QATT

RNI(4) = -QREA

RNI(5) = -IM00

RNI(6) = -VM00+SQRT(VR00*VR00+VX00*VX00)

RNI(7) = -FDIP

During transient simulations,

RNI(3) = 0

RNI(4) = 0

RNI(5) = 0

RNI(6) = SQRT(VR00*VR00+VX00*VX00)

RNI(7) = 0

where,

QBASE = H0*W0 (model power basic value)

YBASE = VNOM**2/QBASE (basic admittance)

GNOM = YBASE*PNOM/VNOM**2 (basic conductance)

BNOM = YBASE*QNOM/VNOM**2 (basic susceptance).

 

If complex power output QAPP is different from zero,

FDP0 = ABS(QATT)/QAPP

otherwise

FDP0 = 0.
