MODULE SBAR

CARD A

Model Generalities

CONNECTION BUS BARS

 

The SBAR module simulates the operation of a bus bar having N branches connected.

Equations used are:

1) Kirchoff equation at branching points

2) Voltage module equation

3) Current module equation

The equations of the SBAR module are written according to the coding of users, that is considering arriving currents as positive.

MODULE SBAR

CARD B

Topological data

INPUT - OUTPUT SCHEME

 

N = number of branches connected to the bus bar (maximum 40)

i = 1,2, ... N

 

 

 

PHYSICAL MEANING OF THE MODULE INPUT-OUTPUT VARIABLES

[V] VR00 real phase voltage at bus bar
[V] VX00 imaginary phase voltage at bus bar
[V] VMOD voltage between lines at bus bar in module
[A] IM_i current of branch i in module
[A] IR_i real current of branch i
[A] IX_i imaginary current of branch i
[ ] SCTO  short circuit condition:
SCTO = 1 bars in short-circuit,
SCTO = 0 bars in normal operation conditions
[ohm] ZRCT short circuit resistance
[ohm] ZXCT short circuit reactance
[ ] SISO isolated bars state:
SISO = 1 isolated bars
SISO = 0 connected bars

MODULE SBAR

CARD C

Physical and Geometrical Data

In addition to the initial values of input and output variables of the SBAR module, whose meaning has been defined in card B, the user must assign the following set of data to complete the input required by the module:

[V] VBASE basic voltage between lines of system in p.u.
[W] QBASE basic power of system in p.u.

Residuals

The equations of the mathematic model, mentioned in Card A, give the following residuals:

These residuals are used in case of normal operation, that is without short-circuits or isolated bars. In presence of short-circuits or of isolated bars the two residuals vary and the different cases are indicated in the module FORTRAN.

In addition to the two above mentioned residuals there are residuals giving the module of the bus bar voltage between lines and the current modules of each different branch.
