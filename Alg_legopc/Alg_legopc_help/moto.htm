MODULE MOTO

CARD A

Model Generalities

ASYNCHRONOUS MOTOR

 

A three order induction motor model is implemented in the MOTO module.

In this representation, the stator transients are neglected, an assumption generally accepted in stability studies and essential to ensure compatibility with the models used for representing other system components, particularly the transmission network.

The module can be used according to twodifferent modalities:

"VELOCITA’ MECCANICA: 0-->INGRESSO 1-->USCITA"

(rotational speed) (input) (output)

In the latter case, the mechanical equation is taken into account within the residuals of the module

Physical and Geometrical Data

The data of the module are:

[-] SDATI data input option
[-] POLI number of poles
[W] QNOM rated power
[Hz] FNOM rated frequency
[V] VNOM rated stator voltage
[V] VBASE system basic voltage
[A] INM rated current
[%] SCORN rated slip speed of the rotor
[-] PU p.u./absolute values option (1à in p.u.; 0à absolute)
[s] TA starting time
[-] COEFF friction coefficient
[A] I0 no-load current
[-] COSFI0 no-load power factor
[A] ICC short circuit current
[-] COSFICC short circuit power factor
[W] QPERD mechanical losses
[W] QPFER losses at rated voltage
[W ] RS stator resistance
[-] RXDSXDR stator and rotor reactance ratio
[W ] XM magnetising reactance
[W ] XDS stator reactance
[W ] XDR rotor reactance
[W ] RRSN stator resistance at rated slip
[W ] RRS1 stator resistance at unitary slip
[p.u.] XM_PU magnetising reactance in p.u.
[p.u.] XDS_PU stator reactance in p.u.
[p.u.] XDR_PU rotor reactance in p.u.
[p.u.] RS_PU stator resistance in p.u.
[p.u.] RRSN_PU stator resistance in p.u. at rated slip
[p.u.] RRS1_PU stator resistance in p.u. at unitary slip

TA and COEFF are defined only in case of MECC=1.

During the data input it is possible to choose one of three different acquisition data modalities:

SDATI=1, if measurement data are known;

SDATI=2, if the parameters of the equivalent circuit are known in S.I. units;

SDATI=3, if the parameters of the equivalent circuit are known in per unit.

 

In case b), the program transforms the data in p.u. values:

XM_PU=XM/ZNOM

XDS_PU=XDS/ZNOM, and so on,

where ZNOM=(VNOM2/QNOM).

In case a), data are dealt with conforming at the IEEE standards (ANSI/IEEE Std 112-1978 – revision of IEEE Std 112A-1964).

The default value for SDATI is 2.

Other default values are the following:

QNOM < 250 MVA QNOM>250 MVA
XDS 0.1*ZNOM 0.1*ZNOM
XDR 0.1*ZNOM 0.1*ZNOM
RS 0.04*ZNOM 0.005*ZNOM
RRS1 0.04*ZNOM 0.03*ZNOM
RRSN 0.04*ZNOM 0.03*ZNOM
XM 4.5*ZNOM 3.5*ZNOM
QPFER 0.01*QNOM 0.015*QNOM

 RXDSXDR = 1 (only if SDATI=1),

and RS = (QP0-QPERD-QPFER)/(3*I0**2) (only if SDATI=1),

MODULE MOTO

CARD F 

Residuals

The equations of the mathematical model are translated in the following residuals:

If the module contains the mechanical equation

IF (KREGIM.AND.OMROT.LE.0.0) THEN (steady-state calculation, and motor stopped)

RNI(1) = OMROT

ELSE

IF (OMMA.LE.0.001) THEN

RNI(1) = KE00*CPOLI/TA

ELSE

RNI(1) = (KE00*CPOLI-(QRES/OMROT)-COEFF)/TA

ENDIF

ENDIF

 

If the breaker is closed (STIN=1),

RNI(1+MECC) = -(ED00-(XS-XS1)*IAUX)/T01+EQ00*DELTOM

RNI(2+MECC) = -(EQ00-(XS-XS1)*IAUR)/T01+ED00*DELTOM

If there's a short circuit (SMAL=1)

RNI(3+MECC) = IR00+(VR00*RCT0+VX00*XCT0)/ZCT0**2.

RNI(4+MECC) = IX00+(VR00*RCT0-VR00*XCT0)/ZCT0**2.

otherwise,

RNI(3+MECC) = -IAUR-(VX00/(FREQ/FNOM*XS1) - EQ00/XS1+ +RS*IAUX/(FREQ/FNOM*XS1))

RNI(4+MECC) = -IAUX-(-VR00/(FREQ/FNOM*XS1)+ ED00/XS1+ +RS*IAUR/(FREQ/FNOM*XS1))

RNI(5+MECC) = KE00+ED00*IAUR+EQ00*IAUX

RNI(6+MECC) = QM00-KE00*(OMMA+EPSI)

RNI(7+MECC) = SCOR-(FREQ/FNOM-OMMA)/ /(FREQ/FNOM)

RNI(8+MECC) = VRM0-VR00

RNI(9+MECC) = VXM0-VX00

 

If the apparent power output is different from zero,

FDP1 = QPE1/QSE1

otherwise

FDP1 = 0

 

During steady-state calculations,

RNI (10+MECC) = -FDP0+FDP1

RNI (11+MECC) = -QPE0+VR00*IR00+VX00*IX00

RNI(12+MECC) = -QQE0+VX00*IR00-VR00*IX00

RNI(13+MECC) = -IMMA+IMOD/INM

RNI(14+MECC) = -QPST+RS*(IAUR**2+IAUX**2)

RNI(15+MECC) = -QPRO+(-QPE0-QPFE-QPST)*SCOR

RNI(16+MECC) = -QPFE+QPFER*(VR00**2+VX00**2)

 

For transient simulations

RNI(10+MECC) = FDP1

RNI(11+MECC) = VR00*IR00+VX00*IX00

RNI(12+MECC) = VX00*IR00-VR00*IX00

RNI(13+MECC) = IMOD/INM

RNI(14+MECC) = RS*(IAUR**2.+IAUX**2.)

RNI(15+MECC) = (-QPE0-QPFE-QPST)*SCOR

RNI(16+MECC) = QPFER*(VR00**2.+VX00**2.)

 

If the breaker is open (STIN=0),

RNI(3+MECC) = -IR00

RNI(4+MECC) = -IX00

RNI(5+MECC) = KE00+ED00*IAUR+EQ00*IAUX

RNI(6+MECC) = QM00-KE00*OMMA

RNI(7+MECC) = -SCOR

RNI(8+MECC) = -VRM0+ED00*OMMA

RNI(9+MECC) = -VXM0+EQ00*OMMA

 

During transient simulations,

RNI(1+MECC) = -(ED00-(XS-XS1)*IAUX)/T01

RNI(2+MECC) = -(EQ00-(XS-XS1)*IAUR)/T01

RNI(10+MECC) = 0

RNI(11+MECC) = 0

RNI(12+MECC) = 0

RNI(13+MECC) = 0

RNI(14+MECC) = 0

RNI(15+MECC) = 0

RNI(16+MECC) = 0

 

for steady-state calculations,

RNI(1+MECC) = -ED00

RNI(2+MECC) = -EQ00

RNI(10+MECC) = -FDP0

RNI(11+MECC) = -QPE0

RNI(12+MECC) = -QQE0

RNI(13+MECC) = IMMA

RNI(14+MECC) = -QPST

RNI(15+MECC) = -QPRO

RNI(16+MECC) = -QPFE

 

where

DELTOM = (FREQ/FNOM-OMMA)*OMBA

IMOD = SQRT(IR00**2+IX00**2)

XR = XDR+XM

XS = XDS+XM (XR and XS are parameters of the asynchronous motor equivalent circuit)

 

If the motor is running (OMMA.GT.0.001)

EPSI = 0.01

otherwise

EPSI = 0.

 

If the breaker is closed (STIN=1)

DELTAR = (RRS1 - RRSN)/(1. - SCORN/100.)

RR = RRSN + DELTAR*((SCOR)-SCORN/100.)

T01 = XR/(OMBA*RR)

otherwise,

T01 = XR/(OMBA*RRSN)

 

XS1 = XS-((XM**2)/XR)

QPE1 = (-VR00*IR00-VX00*IX00)

QQE1 = (-VX00*IR00+VR00*IX00)

QSE1 = SQRT(QPE1*QPE1+QQE1*QQE1)

I0R = VRM0*QPFER

I0X = VXM0*QPFER

 

IAUR = IR00+I0R

IAUX = IX00+I0X

Or if there's a short circuit (SMAL=1),

IAUR = 0.

IAUX = 0.

 

RCT0 = .075*QNOM/VNOM**2 (short-circuit resistance)

XCT0 = .075* QNOM/VNOM**2 (short-circuit reactance)

ZCT0 = SQRT(RCT0**2.+XCT0**2.) (short-circuit impedance)

 

IF (FREQ.LT.0.5) FREQ = 0.5
