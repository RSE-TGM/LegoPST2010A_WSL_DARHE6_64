MODULE MINA

CARD A

Model Generalities

OIL HEADER

 

The MINA module simulates the operation of a header to which Ne pipes arrive and from which Nu pipes are branched off. In the header heavy or lubricating oil is flowing.

Main hypothesis introduced in the model are:

- oil flowing in the header, which must be all of the same type, is perfectly mixed;

- oil has a very low and constant compressibility, neither depending on temperature nor on pressure.

- The specific heat and the thermal conductivity of the header wall material are constant;

- Thermal exchange coefficients between oil and wall and between wall and outside environment are constant.

 

Equations forming the mathematical model are:

- oil mass conservation equation;

- oil energy conservation equation

- energy conservation equation of header wall;

- equation for thermal exchange between heavy oil and header wall;

- equation for thermal exchange between header wall and outside environment.

 

The type of oil considered is determined through an appropriate index assigned in input data; it must be the same for the header and all the pipes both entering and coming out.

MODULE MINA

CARD B

Topological data

INPUT-OUTPUT SCHEME

 

Ne = number of pipes entering the header (max. 40)

Nu = number of pipes coming out from header (max. 40)

 

i = 1, 2, ... Ne

j = 1, 2, ... Nu

 

PHYSICAL MEANING OF THE MODULE INPUT - OUTPUT VARIABLES

[Pa] PNAF pressure in header
[K] TNAF temperature of heavy oil in header
[K] TPAR temperature of header wall
[kg/s] WE_i flow rate of oil entering the i-th piping
[K] TE_i temperature of oil entering the i-th piping
[kg/s] WU_j flow rate of oil coming out from the j-th piping
[K] TU_j temperature of oil coming out from the j-th piping
(in case of flow rate inversion)
[K] TEST temperature of environment oustide the header
[W/m²/K] GEST thermal exchange coefficient between the header wall and the external environment

Note 1: During the model topology definition phase the user is requested to indicate the number of intake pipes and of outgoing pipes for the header one wants to simulate.

In particular the following string appears first on the screen:

MASS FLOW RATE HEADER BRANCH POINT

NUMBER OF INCOMING BRANCHES (01 - 40) ?

and, in the case a numeric value out of the range indicated is entered, the following string appears : ERRORE - N.ENT.= followed by the wrong numeric value entered, then the first string re-appears in order to allow entering of a correct value.

Afterwards, the following string appears:

NUMBER OUTGOING BRANCHES (01 - 40) ?

and, possibly, an error message similar to the one previously shown appears.

The information relevant to the number of intaking and outgoing pipes must be also included in the model geometrical data (see Card C).

MODULE MINA

CARD C

Physical and Geometrical Data

In addition to the initial values of input and output variables of the MINA module, whose meaning has been defined in card B, the user must assign the following set of data to complete the input required by the module:

[ ] NU.TU.EN number of pipes entering the header
[ ] NU.TU.US number of pipes coming out from the header
[m³] VOLUME header internal volume
[kg] MASSA.MET total mass of header walls
[J/kg/K] CP.MET specific heat of the material of header walls
[W/m²/K] COND.MET thermal conductivity of header wall thickness
[m] SPESSORE header wall thickness
[m²] SUP. EST. header external surface
[ ] TIPO FLUIDO index defining the type of oil present in the header (for heavy oil classification see the documentation relevant to "heavy oil properties"). Anyway 1-5 are values for heavy oils, 6 for lubricating oil.

MODULE MINA

CARD F 

Residuals

Residuals for the calculation of the permanent steady condition have the following expression:

 

RNI(1) = SUME+SUMU

                                        (norm. fact. W0)

 

RNI(2) = SUMHE-SUME*HNAF-GAMINT*(TNAF-TM)

                                        (norm. fact. W0*H0)

RNI(3) = TPAR - TM

                                        (norm. fact. T0)

with

TM = (GAMINT*TNAF+GAMEST*TEST)/(GAMINT+GAMEST)

and

GAMINT =SUPES* GAMET*GAMI/(GAMET+GAMI)

GAMEST =SUPES* GAMET*GAMES/(GAMET+GAMES)

where

SUPES wall surface (the internal and external surfaces have been considered as equal)
GAMES thermal exchange coefficient between wall and outside environment
GAMI thermal exchange coefficient between wall and heavy oil
GAMET ratio between thermal conductivity and wall thickness
SUME sum of entering flow rates
SUMU sum of flow rates coming out
SUMHE sum of products between flow rates by enthalpies relevant to entering piping
HNAF oil enthalpy
TPAR wall temperature
TEST temperature of external environment

The expression of residuals for the calculation of transient state calculation has not been indicated. It is noticed that in differential equations, time derivatives have been expressed as finished differences, so that the system of algebraic-differential equations has been transformed in a system of algebraic equations.
